# 📘 Application Fractions – Flashcards, Exercices, Quiz & Challenge

Une application éducative moderne et interactive pour apprendre et maîtriser les fractions.  
Développée pour un usage mobile et desktop, elle combine **flashcards**, **exercices intelligents**, **quiz**, **challenge**, et un **système de difficulté progressif**.

---

## 🎯 Objectif

Aider les élèves (12–13 ans et plus) à comprendre et pratiquer toutes les notions essentielles sur les fractions :

- simplification  
- addition / soustraction  
- multiplication / division  
- comparaisons  
- fractions équivalentes  
- conversions décimales  
- fractions d’un nombre  
- expressions complexes  

---

## 🚀 Fonctionnalités principales

### ✔️ 1. Flashcards interactives  
- 21 cartes couvrant toutes les règles importantes  
- Flip animé  
- Raisonnement détaillé  
- Affichage MathJax pour les fractions  

### ✔️ 2. Exercices intelligents  
- Génération aléatoire  
- Explications détaillées  
- Correction instantanée  
- Affichage LaTeX  
- Compatible mobile  

### ✔️ 3. Mode Progression automatique  
Le niveau augmente ou diminue selon les performances :

- 3 bonnes réponses → niveau +1  
- 2 erreurs → niveau -1  
- niveaux de 1 à 5  

### ✔️ 4. Mode Manuel  
L’utilisateur peut choisir directement :

- Niveau 1 → facile  
- Niveau 2 → intermédiaire  
- Niveau 3 → avancé  
- Niveau 4 → expert  
- Niveau 5 → hardcore  

### ✔️ 5. Quiz (10 questions)  
- Score en direct  
- Correction détaillée  
- Résultat final  

### ✔️ 6. Challenge (3 vies)  
- Score illimité  
- 3 erreurs maximum  
- Questions de difficulté dynamique  

---

## 🧠 Système de difficulté (1 à 5)

| Niveau | Description | Types d’exercices |
|--------|-------------|-------------------|
| **1** | Débutant | opérations simples, même dénominateur |
| **2** | Intermédiaire | dénominateurs différents, simplification |
| **3** | Avancé | opérations en chaîne, divisions |
| **4** | Expert | décimaux, fraction d’un nombre |
| **5** | Hardcore | expressions complexes, comparaisons à 3 fractions |
